# About Me

Hi, I'm **Leshipai** — a software developer with a passion for web technologies, security research, and open source.

I write about things I learn and find interesting across:

- **Front-end** development — JavaScript, React, CSS, browser internals
- **Security** — CTF writeups, web vulnerabilities, browser security
- **Back-end** — APIs, databases, system design
- **Algorithms** — problem solving, data structures

## Why this blog?

Writing forces me to understand things deeply. If I can explain it clearly, I actually understand it. If I can't, I go back and learn more.

This blog is my public notebook — messy at times, but honest.
